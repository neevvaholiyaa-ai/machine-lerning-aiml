# 📊 Tourism in India — Power BI Dashboard

## Overview
This interactive Power BI dashboard visualizes India's tourism trends from 1988 to 
2019, combining demographic, seasonal, and geographic insights to support 
data-driven decision-making for tourism stakeholders.

## Key Components:

- **KPI Cards**: Total FTAs (22M), Total Arrivals (114.21M), India's Global Rank (21)
- **Age-Group Distribution Trend**: Tracks tourist arrivals across age groups (15-24, 25-34, 35-44) from 2000-2019
- **Quarterly Trend (Apr-June)**: Seasonal fluctuation patterns over the years
- **FTAs vs NRIs Arrivals Trend**: Growth comparison between foreign and NRI arrivals
- **Top 5 Region Distribution**: Western Europe (30.38%) and South Asia (23.74%) lead as top source regions
- **Top 5 Countries by Arrivals (2019)**: Bangladesh, U.K., U.S.A, Australia, Canada
- **Purpose of Visit**: Leisure (51.68%) is the dominant travel motivation
- **Year Slicer**: Filter data interactively from 1988 to 2019

## Business Value
This dashboard enables stakeholders to quickly identify peak tourist segments, top 
source regions/countries, and travel motivations — supporting targeted marketing, 
seasonal planning, and resource allocation strategies.

## Tools Used
Power BI Desktop, DAX, Power Query
